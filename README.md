# angular-direcives

> angular-direcives

> 使用angular编写的消息提醒，确认框，时间日程表（考勤记录表）插件


* angular
* moment
* angular-ui-router
* babel
* es6

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start
```

### 时间日历插件

> 结合moment时间插件，方便快速的计算时间； 对应的api地址: http://momentjs.cn/docs/

> 分组当月时间，显示每天对应的事件；
